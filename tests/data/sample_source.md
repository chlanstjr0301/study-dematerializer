## Compactness

A subset K of a metric space X is said to be **compact** if every open cover
of K contains a finite subcover. This is the topological definition of compactness.

The **Heine-Borel theorem** states that in ℝⁿ, a set is compact if and only if
it is closed and bounded.

An open cover of K is a collection of open sets whose union contains K.
A finite subcover is a finite sub-collection that still covers K.

### Sequential Compactness

A set K is sequentially compact if every sequence in K has a convergent
subsequence whose limit is also in K. In metric spaces, compactness and
sequential compactness are equivalent.

### Limit Points

Every infinite subset of a compact set has a limit point in the set.
This property (sometimes called the Bolzano-Weierstrass property) is
equivalent to compactness in metric spaces.

### Connectedness

A set is connected if it cannot be written as the union of two disjoint
nonempty open sets.

### Examples and Counterexamples

The interval [0, 1] is compact (closed and bounded in ℝ, Heine-Borel).
The interval (0, 1) is NOT compact: the open cover {(1/n, 1)} has no
finite subcover because any finite sub-collection fails to cover points
near 0.

A perfect set is both closed and every point is a limit point.
